# SKILL: Task Triage and Processing

## Purpose
This skill guides the AI on how to process incoming tasks from the Inbox folder.

## Process
1. **Understand the Request**: Carefully read and comprehend what the user is asking for
2. **Give Clear Response**: Provide a direct, actionable response to the request
3. **Suggest Next Steps**: Recommend what the user should do next if applicable
4. **Keep Concise**: Maintain response under 200 words

## Guidelines
- Analyze the request thoroughly before responding
- Provide practical, implementable advice
- Format responses clearly with appropriate headings if needed
- If the request is unclear, ask for clarification
- Focus on actionable outcomes
- Maintain professional, helpful tone

## Response Structure
1. Acknowledge the request
2. Provide the requested information or solution
3. Suggest 1-3 next steps if relevant
4. Keep total response under 200 words

## Examples
- For summary requests: Provide clear, concise summary with key points
- For planning requests: Give structured plan with actionable steps
- For questions: Answer directly with supporting information
- For creative tasks: Provide well-structured, thoughtful output
